# lscd (Directory Changer)

`lscd` は、カレントディレクトリ内のサブディレクトリを `whiptail` による TUI メニューで選択し、そのディレクトリへ移動した状態で新しいシェルを起動する CLI ツールです。

## 特徴

- **直感的な操作**: 矢印キーとエンターで移動先ディレクトリを素早く選択できます。
- **シェルの継承**: 現在使用しているシェル（bash, zsh等）を自動判別してサブシェルを起動します。
- **視覚的な識別**: サブシェル内ではプロンプトの先頭に黄色の `[lscd]` が付与され、元のシェルと区別しやすくなります。
- **`ls` オプションのサポート**: `-a` を指定して隠しディレクトリを表示するなど、標準的な `ls` のオプションをそのまま利用可能です。

## インストール

システム全体で利用可能にするために `/usr/local/bin` へインストールします。

```bash
sudo ./install.sh
```

## アンインストール

```bash
sudo ./uninstall.sh
```

## 使い方

```bash
# メニューを起動
lscd

# 隠しディレクトリも含めて表示
lscd -a

# ヘルプとバージョンの表示
lscd --help
lscd --version
```

### サブシェルの終了

サブシェルから元のディレクトリ・元のシェルに戻るには、`exit` を入力するか `Ctrl+D` を押してください。
