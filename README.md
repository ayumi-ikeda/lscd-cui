# lscd
ディレクトリチェンジャー

カレントディレクトリのサブディレクトリをメニュー表示し、選択したディレクトリに移動して終了するシェルスクリプト製のコマンドです。
メニュー表示にはwhiptailが使用されています。

```bash
ls --help
Usage: lscd [ OPTION ]

  -h, --help       使い方を表示して終了する
  -v, --version    バージョン情報を表示して終了する
```

このコマンドは、２バイトコードの文字で書かれているディレクトリにcdしたいときに、入力モードを切り替えずにcdを行いたい場合に有効です。
また、似たような長いディレクトリ名が複数存在する場合にも視覚的かつ直感的にcdできるといったメリットがあります。
